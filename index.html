Range Slider with Popover

convert HTML5 range input field into an interactive slider

<input type="range" />

Basic usage:

1. Include the CSS file.

or copy and paste the CSS code from this file within your project.

<link href="dist/range-slider.min.css" rel="stylesheet" />

2. Auto-Initialize (optional).

add this code to auto-initialize the range slider for all the input type=range elements. and then no need to initialize the slider manually by adding the range-slider class or calling the rangeSlider() function.

<script>window.rangeSliderAutoInit = true;</script>

3. Include the JS file.

<script src="dist/range-slider.min.js"></script>

Initialize the range slider:

If auto-initialization code is not being used, use HTML or JavaScript to initialize it.

via HTML

add the class range-slider to your input type=range element.

<input type="range" class="range-slider" />

via JavaScript

call the rangeSlider() function with a selector for your input type=range element.

rangeSlider('.range-custom-class');
// 'range-custom-class' is the class of the input type=range element. 
// You can also use an ID or any other selectors.
HTML tag example: 
<input type="range" class="range-custom-class" />

or

can use input[type="range"] selector for all the input type=range elements. 

rangeSlider('input[type="range"]');

Advanced Usage:

Hide Popover

To hide the popover, add class range-slider-no-popover to your input type=range element.

<input type="range" class="range-slider-no-popover range-custom-class" />

via JavaScript

rangeSlider('input#custom-id', { popover: false });

Callback

To use a callback function, pass it as the third argument to the rangeSlider() function. 

rangeSlider('input#custom-id', {}, (value) => {
// Handle the value change
document.getElementById('ele-id-value').textContent = `Current Value: ${value}`;
});

Change Slider color

To change the color, copy and paste the CSS below in your project and modify the CSS variables:

:root input.range-slider[type="range"],
:root .range-slider-popover {
    --accent-color: #1f9f95;
}
:root input.range-slider[type="range"] {
    --slider-track-color: lightgray;
}
